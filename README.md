# Geospatial Data Analysis

## Project Overview

This project analyzes location-based sales data to identify optimal regions for business expansion. The analysis focuses on geographical sales performance, demand levels, and business presence to discover high-potential markets.

## Objective

* Analyze location-based sales data
* Identify high-demand regions
* Detect areas with low business presence
* Support business expansion decisions
* Generate actionable geographic insights

## Technologies Used

* Python 3.10
* Pandas

## Project Structure

├── sales_location_data.csv

├── task4.py

├── geospatial_report.csv

└── README.md

## Features

* Sales data analysis
* Regional demand assessment
* Business presence evaluation
* Expansion opportunity identification
* Automated report generation

## Dataset Description

The dataset contains:

* Region Name
* Latitude
* Longitude
* Sales Revenue
* Demand Level
* Business Presence
